# Problem
Patients during recovery often have doubts but cannot frequently contact doctors. This leads to confusion, missed medications, and improper care.
# Healify(Solution)
AI-powered Patient Recovery Assistant helps patients during recovery by answering queries, suggesting basic care plans, and providing medication reminders. It also stores chat history for future reference. The system improves patient recovery by offering simple, continuous support and making post-treatment care more manageable.
# Features
- Personalized greeting using patient name
- AI-based responses to user queries
- Recovery care plan suggestions
- Medication reminder system
- Chat history tracking
# Tech Stack
- Python
- Flask
- Basic AI logic (used in current prototype)
- OpenAI API (for AI response generation - planned integration)
- HTML&CSS
# Future Scope
- Integration with OpenAI API for real-time intelligent responses
# How to Run
1. Download the project files  
2. Open the folder in terminal or command prompt  
3. Install dependencies:
   pip install -r requirements.txt  
4. Run the program:
   python app.py
# Imapct Model
Assumptions:
-1 doctor handles 50 patients/day
-30% queries are repetitive
-Each query takes ~5 minutes
Result:
-Saves ~1.25 hours/day per doctor
Additional Impact:
-Reduces missed medication doses
-Minimizes unnecessary doctor visits
-Improves patient recovery and confidence
